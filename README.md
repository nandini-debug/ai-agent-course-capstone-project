# 🛡️ ScamShield AI

## Multi-Agent Digital Trust Investigator

ScamShield AI is a multi-agent system designed to detect suspicious and potentially fraudulent messages.

The platform helps users identify:

* Fake Job Scams
* UPI Payment Scams
* Phishing Attacks
* WhatsApp Scams
* Lottery Scams

---

## Architecture

User Message
↓
ThreatAgent
↓
EvidenceAgent
↓
RiskAgent
↓
SafetyAgent
↓
Final Verdict

---

## Agents

### ThreatAgent

Detects suspicious keywords and indicators.

### EvidenceAgent

Collects supporting evidence and classifies scam types.

### RiskAgent

Calculates risk score and generates verdict.

### SafetyAgent

Provides security recommendations.
